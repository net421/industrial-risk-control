# CV Project Summary

**Industrial Research Automation Lab**

Built a reproducible Monte Carlo research platform for near-critical industrial
systems using Python, NumPy, and parallel Numba kernels. Implemented deterministic
seed hierarchies, exact reference-versus-optimized validation, atomic checkpoints,
resume support, statistical intervals, SHA-256 artifact manifests, and compact
CSV/JSON reporting. Added GitHub Actions CI, security scanning, scheduled smoke
validation, manual bounded experiments, release automation, and Ubuntu/Ansible
deployment examples.

**Keywords:** Python, Numba, Monte Carlo, CI/CD, GitHub Actions, Linux, Ansible,
testing, reproducibility, performance engineering, research automation.
